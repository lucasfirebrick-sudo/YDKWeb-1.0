# 产品页面标准配置

## HTML基础结构
```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>[Product Name] - [English Title] | High Quality Refractory Materials</title>
```

## CSS引用标准顺序
```html
<!-- 样式表 -->
<link rel="stylesheet" href="../css/core-base.css?v=2.1">
<!-- YDK独立组件 -->
<link rel="stylesheet" href="../css/ydk-navbar.css">
<link rel="stylesheet" href="../css/ydk-footer.css">
<!-- 报价向导组件 -->
<link rel="stylesheet" href="../css/quote-wizard.css">
<!-- 全站浮动按钮 -->
<link rel="stylesheet" href="../css/components/floating-buttons.css">
<!-- 现代化产品详情页专用样式 -->
<link rel="stylesheet" href="../css/product-detail-modern.css">
<link rel="stylesheet" href="../css/product-placeholder.css">
<link rel="stylesheet" href="../css/multi-image-gallery.css">
<link href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.0.0/css/all.min.css" rel="stylesheet">
```

## 图片配置规则

### 普通产品页面
```html
<img src="../images/products/[product-specific-image].png"
     alt="[product-name]"
     class="main-image"
     loading="lazy"
     data-images="[image1],[image2],..."
     onerror="this.onerror=null; this.src='../images/products/placeholder.jpg';"/>
```

### 保温材料页面（特殊配置）
```html
<img src="../images/products/placeholder.jpg"
     alt="保温材料"
     class="main-image"
     loading="lazy"
     data-placeholder="true"
     data-locked="true"
     data-note="保温材料无专用原始图片，不可使用热保温砖图片"
     onerror="this.onerror=null; this.src='../images/products/placeholder.jpg';"/>
```

## 关键差异说明

1. **语言标记**: `lang="en"` (不是 zh-CN)
2. **CSS顺序**: quote-wizard.css 在 floating-buttons.css 之前
3. **必须包含**: 所有CSS和JS文件都必须完整引用
4. **路径统一**: 所有相对路径必须正确（../css/, ../js/, ../images/）
